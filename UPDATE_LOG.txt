DotaFarm — апдейт 2026-05-27 (pairing-ux-v2.1)

— Добавлена кнопка SETUP CREDENTIALS в Pairing-панели (видна когда
  relay-параметры ещё не настроены). Modal с input fields для всех
  необходимых полей + кнопка GENERATE для случайного pair_secret.
  Первичная настройка master-стенда теперь полностью через GUI,
  редактирование farm.json больше не нужно.

DotaFarm — апдейт 2026-05-27 (pairing-ux-v2)

— Новая система парного запуска двух стендов: Pair Code + синхронный
  старт. Включается опцией pairing.uxV2=true в farm.json. По умолчанию
  выключено — старая ферма (один стенд) работает как раньше.

— Master нажимает GENERATE PAIR CODE → получает короткий код → передаёт
  напарнику. Тот нажимает PASTE PAIR CODE → вставляет → оба orchestrator
  автоматически коннектятся к одному relay-каналу. Без правки farm.json
  вручную, без копирования секретов.

— Кнопка START FARM теперь видит состояние партнёра. Пока второй стенд
  не подключён или его heartbeat устарел — кнопка disabled, подсказка
  объясняет что делать. Жмёшь — у партнёра всплывает окно "Peer хочет
  стартовать", он ACCEPT/DECLINE. Оба стенда стартуют одновременно.

— Pairing-панель в GUI показывает: статус (PAIR READY / WAITING PEER /
  STALE LINK / AUTH FAILED / ...), последний heartbeat в ms, RTT,
  счётчики сообщений, uptime связи, состояние sync-start handshake.
  Кнопки RECONNECT / DISCONNECT работают в runtime без рестарта.

— Crash recovery (от предыдущего апдейта) остаётся включён по умолчанию.
  Краши ботов теперь восстанавливаются надёжно, ферма выживает.
