# DotaFarm

Софт для автоматической игры в Dota 2 на нескольких аккаунтах с одной машины.

## Функциональность

**Параллельные инстансы.** До 5 копий Dota 2 одновременно. Каждая работает с отдельным Steam-аккаунтом в изолированной директории профиля.

**Группа из 5 ботов в одной пати.** Один аккаунт назначается лидером, четверо приглашаются автоматически. Заполняют команду 5×5 без рандомов.

**Подбор матчей.** Регион и режим игры задаются в конфиге (turbo, unranked, ranked, captains mode и пр.). Очередь общая для всех инстансов.

**Игровое поведение.** Бот выбирает героя из настроенного пула, идёт на свою линию по назначенной роли, фармит крипов и нейтралов, покупает айтемы по предустановленной сборке. Поддерживаются роли carry, mid, offlane, soft support, hard support.

**Стратегии матча.** Три режима, переключаются в GUI:
- `WIN` — играть на победу
- `LOSE` — играть на поражение (для калибровки smurf-аккаунтов)
- `DEBOOST` — целенаправленный слив для понижения ММР

**Reconnect.** Если бот теряет соединение или клиент крашится во время матча, оркестратор автоматически возвращает его в активную игру.

**Crash recovery.** Упавший инстанс перезапускается без остановки остальных. Восстанавливаются: Steam-сессия, активный матч (если ещё идёт), конфигурация бота.

## GUI

**Дашборд.** Карточка на каждого бота: текущий герой, HP, состояние конечного автомата, время в матче, состояние сетевого подключения, статистика GC-сообщений.

**RAM-панель.** Сверху окна. Шкала загрузки физической памяти, разбивка по standby / modified / committed, суммарный working set всех инстансов фермы. Кнопка ручной очистки + опциональный автозапуск очистки при превышении порога загрузки (по умолчанию 85%).

**Tile.** Раскладка окон Dota по сетке монитора. Доступные варианты: `grid 3+2`, `grid 2+2+1`, `strip 1×5`, `cascade`.

**Pause / Resume.** Глобальная пауза всей фермы. Боты останавливаются на месте, не выходят из матча.

**Логи.** Текстовый лог действий каждого инстанса в реальном времени.

## Сеть и аккаунты

**Per-account proxy.** Поддерживается SOCKS5. Форматы строки подключения: `user:pass@host:port`, `host:port:user:pass`, `host:port`. Каждый Steam-инстанс направляется через свой прокси независимо от остальных.

**Per-account окружение.** Каждый Steam запускается в собственной директории. Конфиги, кэш, библиотеки игр, авторизационные токены изолированы между инстансами.

**Региональная привязка.** Регион сервера Dota задаётся в настройках.

## Требования

- Windows 10/11 x64
- ≥32 GB RAM (рекомендация при 5 инстансах)
- Любая GPU с поддержкой DirectX 11
- Стабильное интернет-соединение
- Steam-аккаунты с активированной Dota 2 (один раз требуется ручной логин для инициализации профиля)
- Прокси-сервера (опционально, но без них все инстансы делят один IP)

## Лицензия

Привязывается к железу при первой активации. Одна лицензия — одна машина.
